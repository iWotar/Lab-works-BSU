# Лабораторная 2

### Сделанные пункты
* Создан интерфейс ```Shape```, которого имплементируют 3 вида фигур ```Cube```, ```Pyramid```, ```Sphere```. У них реализованы перегрузки функций получения данных о размерах, типе, объёме и получения относительных координат для рисования *(само рисование пока не реализовано)*.
* Реализован класс ```Bag``` для хранения фигур. При добавлении фигуры сортируются по размеру, а так же проверяется превышение объема вместимости. При проверке объема кидается ```CapacityExeption```, но в данной реализации он перехватывается в методах добавления и лог выводится в консоль, фигура же не добавляется.
* Реализован GUI для работы с приложением. В нем имеется две кнопки *Add shape*, *Delete shape*, при нажатии на которые открывается диалоговое окно с полями для указания данных. Так же в начальном окне имеется надпись с указанием оставшегося объема и зона, куда выводятся данные о фигурах. Эти данные включают в себя: индекс, тип, объем и размеры фигуры. В поле вывода реализовано прокручивание при невлезании в окно. Все реализовано на Layout, поэтому поддерживает изменение размера окна.
* При добавлении сначала появляется окно для выборы фигуры, а после выбора появляется окно для ввода данных через запятую, данными являются вещественные числа. Ввод проверяется на валидность и, в случае несоответствия, показывается диалоговое окно с ошибкой. Аналогично при удалении, только сразу пояплявтся окно для ввода индексов.
